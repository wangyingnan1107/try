## Python vs Java 语法对照

### 1. 基本语法差异

| Python | Java | 说明 |
|--------|------|------|
| `def method(self):` | `public void method() {}` | 方法定义 |
| `self.var = value` | `this.var = value;` | 实例变量 |
| `if not var:` | `if (var == null || var.isEmpty())` | 空值检查 |
| `for item in list:` | `for (Item item : list)` | 循环 |
| `list.append(item)` | `list.add(item)` | 添加元素 |
| `dict['key']` | `map.get("key")` | 字典访问 |
| `f"文本{var}"` | `String.format("文本%s", var)` | 字符串格式化 |

### 2. Python特有语法

#### 列表推导式
```python
# Python
lines = [line.strip() for line in text.split('\n') if line.strip()]

# Java等价写法
List<String> lines = new ArrayList<>();
for (String line : text.split("\n")) {
    String trimmed = line.trim();
    if (!trimmed.isEmpty()) {
        lines.add(trimmed);
    }
}
```

#### 生成器表达式
```python
# Python
count = sum(1 for c in text if c.isalpha())

# Java等价写法
int count = 0;
for (char c : text.toCharArray()) {
    if (Character.isLetter(c)) {
        count++;
    }
}
```

#### 切片操作
```python
# Python
text[:500]  # 前500个字符
text[1:10]  # 从索引1到10
text[-1]    # 最后一个字符

# Java
text.substring(0, 500)
text.substring(1, 10)
text.charAt(text.length() - 1)
```

#### 字典推导式
```python
# Python
new_dict = {k: v for k, v in old_dict.items() if k in keep_list}

# Java
Map<String, Object> newMap = new HashMap<>();
for (Map.Entry<String, Object> entry : oldMap.entrySet()) {
    if (keepList.contains(entry.getKey())) {
        newMap.put(entry.getKey(), entry.getValue());
    }
}
```

### 3. 类型提示（可选）

```python
# Python类型提示（可选，主要用于IDE）
def method(url: str) -> str:
    return url

# Java（强制类型）
public String method(String url) {
    return url;
}
```

### 4. 异常处理

```python
# Python
try:
    result = risky_operation()
except SpecificError as e:
    logger.error(f"错误: {e}")
except Exception as e:
    logger.error(f"其他错误: {e}")
finally:
    cleanup()

# Java
try {
    String result = riskyOperation();
} catch (SpecificException e) {
    logger.error("错误: " + e);
} catch (Exception e) {
    logger.error("其他错误: " + e);
} finally {
    cleanup();
}
```

### 5. 装饰器

```python
# Python @staticmethod装饰器
@staticmethod
def static_method():
    pass

# Java static关键字
public static void staticMethod() {
    // ...
}
```

### 6. None vs null

```python
# Python
if value is None:
    return None

# Java
if (value == null) {
    return null;
}
```
